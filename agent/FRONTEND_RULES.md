# FRONTEND_RULES.md
## UI/UX & Responsive Rules

### Philosophy
- Mobile-first, not mobile-only
- Tablet & Desktop are enhanced layouts

### Breakpoints
xs <480px
sm 480–767px
md 768–1023px
lg 1024–1439px
xl ≥1440px

### Layout
Mobile: 1 column
Tablet: 2 columns
Desktop: main + sidebar (max-width 1320px)

### Ads Rules
- Ads after results
- No popup
- No redirect
